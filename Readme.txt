Prerequisition:

Install Nodejs, if not, go to https://nodejs.org/en/download/ and install it.
Install Apache, mysql. Use Xampp or other to manage
Import file "websocket-midterm.sql" in folder "database" to phpmyadmin
Change .env file to fit your phpmyadmin (host,user,password) settings
Run step:
    npm i
    npm start
